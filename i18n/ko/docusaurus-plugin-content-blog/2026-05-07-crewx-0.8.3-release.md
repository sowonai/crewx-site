CrewX 0.8.3은 **외부 접속 인증**, **Drizzle ORM 기반 데이터베이스 재설계**, 그리고 새로워진 **스레드 탐색 경험**을 제공합니다.

<!--truncate-->

## 🎯 주요 신기능

### 외부 접속 인증

새로운 `BaseAuthGuard`가 외부 네트워크에 노출된 CrewX 인스턴스를 보호합니다. 로컬호스트 접속은 자유롭게 통과하고, 외부 IP는 ID/PW 인증이 필요합니다.

- 로컬 개발 환경에서는 설정 없이 바로 동작
- 외부 IP 접속 시 자동으로 인증 요구
- 환경 변수로 인증 정보 설정

### 데이터베이스 재설계

전체 영속성 레이어를 **Drizzle ORM**으로 재작성했습니다. 타입 안전한 리포지토리와 자동 마이그레이션으로 안정성과 업그레이드 편의성이 크게 향상됩니다.

### 스레드 화면 개편

- **서버 사이드 검색** — 쿼리 파라미터 기반 빠른 스레드 검색
- **플랫폼 필터 & 배지** — CLI, Web, API, Slack별 필터링
- **제목 편집** — 연필 아이콘 클릭으로 스레드 이름 변경
- **한국어 IME 수정** — 한국어 검색 입력 정상 처리

### CLI 관리 화면 개선

CLI 관리 화면에 **OpenCode**가 추가되었습니다. Claude, Codex, Gemini, Copilot과 함께 각 도구를 독립적으로 병렬 업데이트할 수 있습니다.

## 🚀 업그레이드

```bash
npm install -g crewx@0.8.3
```
