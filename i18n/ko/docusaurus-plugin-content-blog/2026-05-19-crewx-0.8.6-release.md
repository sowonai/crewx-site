CrewX 0.8.6은 시작을 더 쉽게 만들었습니다. 아무 디렉터리에서 `npx crewx@latest`만 실행하세요 — 설정 파일 없이 바로 시작합니다.

<!--truncate-->

## 🎯 주요 신기능

### 제로 설정 부팅

첫 대화까지 설정이 필요 없습니다. 아무 디렉터리에서 CrewX를 실행하면, 가이드 화면이 기본 세팅을 안내합니다.

```bash
npx crewx@latest
```

이것만으로 충분합니다. `crewx.yaml` 파일이 없어도 됩니다.

### 기본 에이전트 업그레이드

새 워크스페이스는 기본적으로 `acp/claude` 프로바이더를 사용하며, high-effort 쿼리 모드가 적용되어 더 나은 에이전트 응답을 제공합니다.

### 태스크 UI 개편

태스크 목록과 상세 패널이 새롭게 디자인되었습니다:

- **프로바이더 아이콘** — 어떤 AI 엔진이 처리했는지 한눈에 확인
- **소요 시간** — 초/분/시간 단위의 읽기 쉬운 표시
- **모드 배지** — Execute(빨강) vs Query(회색) 구분
- **버전 정보** — 각 태스크에 사용된 CrewX 버전 표시

### 온보딩 영속 저장

온보딩 팝업을 한번 닫으면 다시 표시되지 않습니다 — 새로고침이나 재방문 시에도 유지됩니다.

### Chrome X 개선

플로팅 버튼이 32px로 작아지고, 평상시 반투명(0.6)으로 표시되며, 전체 화면 모드에서는 자동으로 숨겨집니다.

## 🚀 업그레이드

```bash
npm install -g crewx@0.8.6
```
