CrewX 0.8.5에서 **Chrome X 브라우저 확장**과 새로운 **ACP 프로바이더 시스템**이 출시됩니다. AI 에이전트가 웹 브라우징을 함께 할 수 있고, 에이전트 설정이 훨씬 유연해졌습니다.

<!--truncate-->

> **참고:** v0.8.4는 패키징 버그로 폐기되었습니다. v0.8.5가 동일 기능의 정식 릴리스입니다.

## 🎯 주요 신기능

### Chrome X 브라우저 확장

AI 에이전트가 웹 브라우징을 함께 할 수 있습니다. Chrome X 확장을 설치하면:

- **사이드 패널 채팅** — 브라우징하면서 CrewX 에이전트와 대화
- **브라우저 도구** — 에이전트가 탭 열기, 페이지 읽기, 요소 클릭, 탭 닫기 가능
- **실시간 동기화** — SSE 기반 확장↔서버 통신

### ACP 프로바이더 시스템

새로운 `acp/*` 프로바이더 네임스페이스가 기존 `cli/*` 시스템을 대체합니다:

- **동적 모드 & effort 설정** — UI에서 에이전트별 query/execute 모드 설정
- **모델 선택** — YAML 편집 없이 에이전트별 적합한 모델 선택
- **프로바이더 메타데이터 API** — 사용 가능한 모드, effort, 모델을 런타임에 조회

### 에이전트 편집기 업그레이드

에이전트 편집 폼이 ACP 프로바이더를 동적으로 조회하여, 사용 가능한 모드·effort·모델을 검색 가능한 드롭다운으로 보여줍니다.

### 확장 사고 (Extended Thinking)

Claude의 사고 블록이 파싱되어 스레드 UI에 표시됩니다. 모델의 추론 과정을 직접 확인할 수 있습니다.

### 워크스페이스 URL 라우팅

워크스페이스가 깔끔한 `/ws/:slug` URL을 사용합니다. 모든 페이지가 slug로 구분되어 북마크 가능한 워크스페이스 링크를 지원합니다.

## 🚀 업그레이드

```bash
npm install -g crewx@0.8.5
```
