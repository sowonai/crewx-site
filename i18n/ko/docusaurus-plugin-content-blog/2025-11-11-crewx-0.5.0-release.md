CrewX 0.5.0이 출시되었습니다! 이번 릴리스에서는 두 가지 주요 기능이 추가되었습니다:

1. **Skills 시스템**: Claude Code 스킬과 100% 호환되는 재사용 가능한 AI 능력 시스템
2. **Mention-Only 모드**: Slack에서 멘션했을 때만 응답하는 선택적 Bot 모드

<!--truncate-->

## 🎯 주요 신기능

### 1. Skills 시스템 - Claude Code와 완벽 호환

CrewX에 재사용 가능한 AI 전문 능력을 추가할 수 있는 Skills 시스템이 도입되었습니다. **Claude Code의 Skills 포맷과 100% 호환**되어, 기존 Claude 스킬을 수정 없이 바로 사용할 수 있습니다.

#### 왜 Skills인가?

- **재사용성**: 한 번 만든 스킬을 여러 에이전트에서 활용
- **호환성**: Claude Code 스킬 라이브러리를 그대로 사용
- **범용성**: Claude, Gemini, Copilot, Codex 모든 AI 제공자에서 동작
- **버전 관리**: 스킬을 독립적으로 관리하고 배포

#### 간단한 사용법

**1. 스킬 만들기**

`skills/hello/SKILL.md` 파일을 생성:

```markdown
---
name: hello
description: 친절한 인사를 제공하는 스킬
version: 0.0.1
---

# Hello Skill

이 스킬은 사용자에게 친절한 인사를 제공합니다.

## 기능
- 맞춤형 인사말 생성
- 스킬 시스템 동작 확인
```

**2. crewx.yaml에 설정 추가**

```yaml
# 프로젝트 전체 스킬 설정
skills:
  paths:
    - ./skills              # 커스텀 스킬 디렉토리
  include:
    - hello                 # 사용할 스킬 지정

# 특정 에이전트에 스킬 추가
agents:
  - id: "senior_dev"
    provider: "cli/claude"
    skills:
      include:
        - hello
        - code-reviewer
    inline:
      prompt: |
        You are a senior developer.
```

**3. 사용하기**

```bash
# 스킬이 적용된 에이전트 실행
crewx query "@senior_dev 안녕하세요"

# 여러 AI 제공자에서 테스트
crewx query "@skill_tester_claude test hello skill"
crewx query "@skill_tester_gemini test hello skill"
```

#### Claude Code 스킬 바로 사용하기

Claude Code의 `.claude/skills/` 디렉토리에 있는 스킬을 그대로 활용할 수 있습니다:

```yaml
skills:
  paths:
    - ./.claude/skills    # Claude Code 스킬 경로
    - ./skills            # CrewX 커스텀 스킬
```

수정 없이, 복사 없이, 바로 동작합니다! 🎉

### 2. Slack Mention-Only 모드

Slack Bot이 이제 **mention-only 모드**를 지원합니다. 이 모드를 활성화하면 CrewX Bot이 명시적으로 멘션되었을 때만 응답합니다.

#### 왜 필요한가?

기본 모드에서는 Bot이 채널의 모든 쓰레드에 자동으로 응답합니다. 하지만 팀원끼리 대화하는 쓰레드나 간단한 질문에는 AI가 끼어들 필요가 없을 때가 많습니다.

#### 사용법

```bash
# Mention-only 모드로 Slack Bot 실행
crewx slack --mention-only

# 일반 모드 (모든 쓰레드에 응답)
crewx slack
```

**동작 방식:**

- **일반 모드**: Bot이 모든 쓰레드의 메시지에 자동 응답
- **Mention-only 모드**: `@CrewX` 멘션이 있을 때만 응답

```
# Mention-only 모드에서
👤 "코드 리뷰 부탁해"           → Bot 응답 없음
👤 "@CrewX 코드 리뷰 부탁해"   → Bot 응답! ✅
```

이제 필요할 때만 AI를 호출할 수 있어, 더 효율적인 팀 협업이 가능합니다.

## 🚀 업그레이드 방법

```bash
# NPM으로 업데이트
npm update -g crewx

# 또는 재설치
npm install -g crewx@latest

# 버전 확인
crewx --version  # 0.5.0이 출력되어야 합니다
```

기존 `crewx.yaml` 설정은 그대로 동작합니다. Skills를 사용하려면 위에서 설명한 대로 `skills` 섹션을 추가하면 됩니다.
