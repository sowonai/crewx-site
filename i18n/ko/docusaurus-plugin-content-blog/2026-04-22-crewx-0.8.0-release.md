CrewX 0.8.0이 출시되었습니다! 이번 릴리스의 핵심은 AI 에이전트를 Slack, Google Chat, Teams에 배포할 수 있는 **채널 어댑터**와, 멀티 프로젝트를 관리하는 **워크스페이스** 기능입니다.

<!--truncate-->

## 🎯 주요 신기능

### 채널 어댑터

AI 에이전트를 메시징 플랫폼에 간단히 배포할 수 있습니다. **Slack**, **Google Chat**, **Microsoft Teams** 어댑터를 기본 제공합니다.

```bash
# Slack 봇 모드로 실행
crewx slack --agent @my-agent
```

각 어댑터가 플랫폼별 인증, 이벤트 라우팅, 메시지 포맷팅을 처리하므로 에이전트 설정은 플랫폼에 관계없이 동일하게 유지됩니다.

### 워크스페이스 관리

워크스페이스로 AI 에이전트 프로젝트를 체계적으로 관리하세요. 각 워크스페이스는 독립적인 에이전트, 스레드, 사용량 추적을 지원합니다.

- UI에서 워크스페이스 생성 및 전환
- 워크스페이스별 사용량 한도 및 과금
- 빠른 설정을 위한 초기화 다이얼로그

### 대화 스레드

스레드에 **플랫폼 네임스페이스**가 적용되어, CLI·Web·Slack·API 대화가 구분되어 관리됩니다. 대화 모드(query/execute)도 세션 간 유지됩니다.

### UI 개선

- **활동 로그** — 3줄 프리뷰 + 전체 화면 상세 모달
- **채팅 입력 바** — 인라인 query/execute 모드 선택
- **디렉터리 피커** — UI에서 디렉터리 탐색 및 선택
- **문서 편집** — 브라우저에서 마크다운 문서 직접 편집

## 🚀 업그레이드

```bash
npm install -g crewx@0.8.0
```

> **참고:** 같은 날 Windows 호환성 수정이 포함된 v0.8.1이 배포되었습니다. Windows 환경이라면 `crewx@0.8.1`을 설치하세요.
